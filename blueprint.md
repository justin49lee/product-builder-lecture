# Lotto Number Generator

## Overview

This is a simple web application that generates 5 random lottery numbers between 1 and 45.

## Design and Features

*   **UI:** A clean and simple interface with a button to generate numbers and a display area for the results.
*   **Logo:** A simple and modern logo that represents the app's purpose.
*   **Colors:** A vibrant color scheme to create an energetic look and feel.
*   **Typography:** Clear and readable fonts to enhance user experience.
*   **Interactivity:** A button with a "glow" effect on hover.
